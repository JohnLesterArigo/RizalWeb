# The Shaping of a Hero: José Rizal's Character Development

A static HTML and CSS educational website about the biological, environmental, and life-experience factors that shaped José Rizal's character and heroism. Built for simple deployment on Vercel.

## Deploy on Vercel

1. Upload this folder to a GitHub repository.
2. Open Vercel and choose **Add New Project**.
3. Import the repository.
4. Use the default settings. No build command is needed.
5. Deploy.

The main page is `index.html`, and the styling is in `styles.css`.

The About Me page uses `Image/lester.jpg` for the profile photo.
